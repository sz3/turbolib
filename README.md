# turbolib

General purpose C++11 support library for turbopump.

### Build dependencies

* linux (for now)
* g++ >= 4.8 (for proper c++11)
* cmake
* may or may not compile on 64-bit until I get an automated build box set up ¯\_(ツ)_/¯

### Library dependencies

* UDT (libudt)
* intel's TBB library (libtbb)
* boost::uuid
